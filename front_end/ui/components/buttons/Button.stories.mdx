# Buttons

### &lt;devtools-button>

## Variants

```
data.variant = Variant.Primary
```

Prefer the data= setter over the individual setter for better typesafety.

<Canvas>
  <Story name="Default" args={{ data: { variant: 'primary' } }}>
    {Template.bind({})}
  </Story>
</Canvas>

## IconButton

Use the variant `Variant.Icon` and either `iconName` or the `iconUrl` to use an IconButton.
When using the icon variant, there mustn't be a slot (inner HTML content).

<Canvas>
  <Story name="IconButton" args={{ data: { variant: 'icon', iconName: 'plus' } }}>
    {Template.bind({})}
  </Story>
</Canvas>
